What is AWS
AWS stands for Amazon web services and it is a cloud provider.
It provides you with servers and services that you can use on demand and scale easily.

AWS Regions 
• AWS has Regions all around the world 
• Names can be us-east-1, eu-west-3
• Most AWS services are region-scope

AWS Availability Zones
• Each region has many availability zones 
(usually 3, min is 3, max is 6). Example:
• ap-southeast-2a
• ap-southeast-2b
• ap-southeast-2c
• Each availability zone (AZ) is one or more 
discrete data centers with redundant power, 
networking, and connectivity
